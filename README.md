# Vite + Vue3 + js + vue-router + pinia + element-plus + axios + less

本项目基于Vite + Vue3 + js + vue-router + pinia + element-plus + axios + less创建



运行方式

```text
cd my-project
npm i
npm run dev
```



需要修改的地方

```text
utils/request.js
stores/token.js
router/index.js
```



可以修改的地方

```text
vite.config.js
```

该文件中的`server.proxy`,`server.host`,`server.port`可以进行更改,其他地方不建议更改



可以添加内容的地方

```text
api/index.js
```
